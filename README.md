# Usage

<!-- usage -->

```sh-session
$ npm install -g druidctl
$ druidctl COMMAND
running command...
$ druidctl (--version)
druidctl/0.0.1-alpha.1 linux-x64 node-v18.20.4
$ druidctl --help [COMMAND]
USAGE
  $ druidctl COMMAND
...
```
